# Telegram Video Watermark Bot

Automatically detects and replaces watermarks on video thumbnails.

## Features
- 🔍 Automatic watermark detection
- 🎨 Smart content-aware removal  
- ✨ Beautiful new watermarks with outline
- 💾 Per-user settings

## Deployment

### On Render.com

1. Fork this repository
2. Create a new Web Service on Render
3. Connect your GitHub repository
4. Set Environment Variables:
   - `BOT_TOKEN`: Your Telegram bot token
5. Deploy!

## Commands
- `/start` - Show instructions
- `/setwatermark <text>` - Set watermark text
- `/setposition <pos>` - Set position
- `/settings` - View settings
- `/reset` - Reset settings
